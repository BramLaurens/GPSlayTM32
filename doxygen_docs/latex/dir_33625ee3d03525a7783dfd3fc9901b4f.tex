 De task \doxylink{admin_8h_af42059fd155a512774c89e7fc38bc03c}{UART\+\_\+keys\+\_\+\+IRQ()} leest de queue uit nadat het LINEFEED-\/character is gevonden (nadat ENTER gedrukt is). De task leest de queue uit en vult een eigen buffer. Deze buffer wordt via Task\+Notify doorgestuurd naar de task die deze buffer interpreteert\+: \doxylink{admin_8h_af909ca669055035da2246051d81ae747}{UART\+\_\+menu()}. \end{DoxyCompactList}\end{DoxyCompactItemize}
